### 【単位の種類】
* **絶対単位**
	他の影響を受けピッカー図、常に同じ大きさになる単位
	例）cm

* **相対単位**
	他の大きさに従って相対的に変化する単位
	例）％


### 【絶対単位の種類】
* **日常的に使用される単位**
	mm(ミリメートル)、cm(センチメートル)
	in(インチ)：2.54cm
	※画面上では1cmの大きさが変化するので、印刷の時にしか利用できない

* **印刷で使用されている単位**
	DDPで使われることが多い
	インチを基準としているので、Webでは使われない
	pt(ポイント)：1/72インチを基準とする単位
	pc(パイカ)：1/6インチを基準とする単位

* **Web制作でよく使われる単位**
	px(ピクセル)
		画面に表示される「グラフィック」最小単位。日本語では「画素」

### 【相対単位の種類】
* **Web制作でよく使われる単位**
	%(パーセント)
		基準値に対する割合を表す。パーツの幅や高さの調整に使う

	em(エム)
		親のフォントサイズを基準に変わる

	rem(レム)
		ルート要素のフォントサイズを基準にする
